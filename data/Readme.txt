==========================================
Solar Power Concentrator Dataset
==========================================

Author: Rahmadi Putra Aji
Affiliation: Universitas Gadjah Mada (TRIK - Instrumentation and Control Engineering)
Dataset Type: IoT Sensor Data (LDR, Voltage, Current)
Date Collected: 3-Day Period (450 records)

Description:
This dataset contains measurements from a vertical two-axis solar concentrator system.
Each record represents sensor readings taken every 2 minutes in both static and tracking modes.

Files:
- data_putra.csv : main dataset file
